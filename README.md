# RIS-Enabled-mmWave-Localization-Reproduction
High-fidelity MATLAB reproduction of RIS-enabled mmWave localization for autonomous vehicles. Features 3D ray-tracing in Sydney CBD, massive MIMO beamforming, and a Lyapunov-stable differential estimator. Achieves 20cm accuracy to complement vSLAM. Based on Eskandari et al. (IEEE 2026) .
